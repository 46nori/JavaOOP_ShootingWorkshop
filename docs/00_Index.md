# 目次

このワークショップでは、以下のステップでオブジェクト指向を学んでいきます。

## 🎮 作成するゲームのイメージ
このワークショップの最終目標は、**「画面に複数の敵キャラクターを配置し、それぞれ異なる動きでアニメーションさせること」**です。

シューティングゲームの「敵キャラクター部分」に焦点を当てて開発します。
- 画面上をまっすぐ進む敵
- 波打つように動く敵

これらをオブジェクト指向の考え方を使って効率よく管理・動作させるプログラムを作成します。

## 📂 フォルダ構成
Eclipseの `src` フォルダの中にステップごとのパッケージが、`docs` フォルダの中に解説ドキュメントが用意されています。

```text
JavaOOP_ShootingWorkshop/
├── src/                   # ソースコード
│   ├── step01_concept/        # クラスの概念（コードはありません）
│   ├── step02_class_basic/    # Enemyクラスの基本実装
│   ├── step03_animation/      # 単体でのアニメーション実行
│   ├── step04_array/          # 多数の敵の生成と制御
│   ├── step05_inheritance/    # 継承を使った敵の種類の追加
│   ├── step06_polymorphism/   # ポリモーフィズムによる一括管理
│   └── step07_encapsulation/  # カプセル化によるデータの保護
│
└── docs/                  # 解説ドキュメント
    ├── 01_Concept.md
    ├── 02_ClassBasic.md
    └── ...
```

## 🎯 全体の目標
- クラス (Class) とインスタンス (Instance) の違いを理解する
- 継承 (Inheritance) を使ってコードを再利用する
- ポリモーフィズム (Polymorphism) を使って柔軟なプログラムを書く
- カプセル化 (Encapsulation) で安全な設計を行う

それでは、[Step 1: クラスの概念](01_Concept.md) から始めましょう！
